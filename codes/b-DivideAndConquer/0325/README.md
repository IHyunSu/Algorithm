# 분할 정복(Divide and Conquer) 예시

분할 정복은 문제를 더 작은 하위 문제들로 나누고, 각각을 재귀적으로 해결한 후, 그 결과를 결합하여 원래 문제를 해결하는 알고리즘 기법입니다.

---

## 1. 스트라센 행렬 곱셈(Strassen's Matrix Multiplication)

스트라센 알고리즘은 행렬 곱셈의 곱셈 횟수를 줄여 연산 속도를 향상시킵니다.

- **분할(Divide)**: 두 행렬을 각각 4개의 하위 행렬로 나눕니다.
- **정복(Conquer)**: 8번이 아닌 7번의 행렬 곱셈을 재귀적으로 수행합니다.
- **결합(Combine)**: 결과를 결합하여 최종 행렬을 구합니다.

시간 복잡도: **O(n^2.81)**

---

## 2. 카라츠바 정수 곱셈(Karatsuba's Integer Multiplication)

카라츠바 알고리즘은 큰 수의 곱셈을 더 빠르게 수행합니다.

- **분할(Divide)**: 숫자를 절반으로 나눕니다.
- **정복(Conquer)**: 절반끼리 곱셈을 3번 수행합니다.
- **결합(Combine)**: 덧셈과 뺄셈으로 최종 결과를 계산합니다.

시간 복잡도: **O(n^1.585)**

---

## 3. 분할 정복을 이용한 피보나치 수(Fibonacci)

피보나치 수열을 행렬 거듭제곱 또는 빠른 분할 방식으로 계산할 수 있습니다.

- **분할(Divide)**: 다음 공식 사용  
  F(2k) = F(k)[2F(k+1) − F(k)]  
  F(2k+1) = F(k+1)^2 + F(k)^2
- **정복(Conquer)**: F(k), F(k+1)를 재귀적으로 계산합니다.
- **결합(Combine)**: 위 식을 이용해 F(n)을 계산합니다.

시간 복잡도: **O(log n)**

---

## 4. 쿼드트리 인코더(Quadtree Encoder)

쿼드트리는 이미지 압축과 표현에 사용됩니다.

- **분할(Divide)**: 이미지를 4개의 영역으로 나눕니다.
- **정복(Conquer)**: 각 영역이 같은 색이면 하나로 표현, 다르면 다시 분할합니다.
- **결합(Combine)**: 인코딩된 데이터를 결합하여 전체 이미지를 표현합니다.

---

## 5. 쿼드트리 디코더(Quadtree Decoder)

쿼드트리 디코더는 인코딩된 트리 구조를 바탕으로 이미지를 복원합니다.

- **분할(Divide)**: 트리 구조를 따라 영역을 나눕니다.
- **정복(Conquer)**: 각 영역을 재귀적으로 복원합니다.
- **결합(Combine)**: 복원된 하위 영역들을 결합하여 전체 이미지를 만듭니다.
