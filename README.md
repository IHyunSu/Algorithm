## 🧪 예시 코드 분석

```cpp
for (i = 1; i <= 2*n; i += 4)
    for (j = 1; j <= 2*m; j *= 2)
        for (k = 4*p; k >= 1; k /= 2)
            cnt++;
```

- 시간복잡도: Θ(n log m log p)
